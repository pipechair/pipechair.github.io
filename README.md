## ラッパンアスクプレイヤーへの公開データ

現在ダンジョンマスターとして運営しているD&D 5eゲーム [Rappan Athuk 5e](https://www.drivethrurpg.com/en/product/262702/rappan-athuk-5e?affiliate_id=3787011)/[Frog god Games](https://www.froggodgames.com/) の記録並びにプレイヤーへの開示可能なデータ・ドキュメントをまとめています。

このサイトのデータは、ウィザーズ・オヴ・ザ・コーストの[ファンコンテンツポリシー](https://company.wizards.com/ja/legal/fancontentpolicy)並びに、[System Reference Document 5.1](https://media.dndbeyond.com/compendium-images/srd/5.1/SRD_CC_v5.1.pdf)を元に作成されています。
